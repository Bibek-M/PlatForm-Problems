This simple program involves receiving inputs for num (the number to be printed) and n (the count indicating how many times num should appear). You need to ensure the numbers are printed in the correct format on one line.

Example
Suppose you receive the following input:

num = 3
n = 5
Your program should output:

Here, the number 3 is printed five times, with each occurrence separated by a single space.
