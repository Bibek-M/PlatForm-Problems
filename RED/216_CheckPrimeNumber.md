Determine if a given integer is a prime number. A prime number is defined as a number greater than 1 that cannot be formed by any other combination of smaller numbers except being multiplied by 1 and itself. In other words, a prime number has no divisors other than 1 and itself.

### Example
Input: num = 29

Output: True

Explanation: 29 is a prime number because it is only divisible by 1 and 29.

Input: num = 35

Output: False

Explanation: 35 is not a prime number because it has divisors 1, 5, 7, and 35.
